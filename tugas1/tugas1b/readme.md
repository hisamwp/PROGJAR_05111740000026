<h2>Tugas 1b</h2>

Client request file ke server.

menjalankan server.py:
```
py server.py
```

menjalankan client.py, serta memilih file yg diminta:
```
py client.py [nama file]
```

Jalankan server terlebih dahulu, baru client.
