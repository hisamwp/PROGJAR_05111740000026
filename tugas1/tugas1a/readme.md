<h2>Tugas 1a</h2>

Mengirim file dari client ke server.

menjalankan server.py:
```
py server.py
```

menjalankan client.py, serta memilih file yg dikirim:
```
py client.py [nama file]
```

Jalankan server terlebih dahulu, baru client.
